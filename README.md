## 単純命令セット
命令セット
* R型 [7:0]opcode, [12:8]rd1, [17:13]rd2, [22:18] rw, [31:23] 空き （計算に使う命令）
* L型 [7:0]opcode==4, [17:8]memory, [22:18] r, [31:23]空き （データメモリからレジスタにデータ書き込みする命令）
* W型 [7:0]opcode==5, [17:8]memory, [22:18] r, [31:23]空き  （レジスタからデータメモリにデータ書き込みする命令）

単純にR型は
* opcode==0: 足し算
* opcode==1: 引き算
* opcode==2: 掛け算
* opcode==3: 割り算
とします。
